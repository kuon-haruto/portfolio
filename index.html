自己紹介

作品 01

線のこちら側 / 向こう側

境界線で世界を分けるパズルアクションゲーム。

ジャンル：パズル 制作人数：4人 担当：企画 / タスク振り分け / 仕様整理 / レベル設計 / 雑用

概要

プレイヤーは自由に境界線を引くことができ、線を境に世界が分かれます。片側は重力あり、もう片側は無重力になります。境界線をうまく配置して、ゴールを目指すパズルアクションゲームです。

企画意図

重力の違いを利用することで、プレイヤーの空間認識と発想力を試すパズル体験を目指した。

工夫した点

  • 重力と無重力の切り替えによる思考性のあるパズル設計
  • 線の引き方によって解法が変わる自由度

作品 02

反転暗殺

盤面反転を使ったステルスアクションゲーム。

ジャンル：ステルスアクション 制作人数：4人 担当：企画 / タスク振り分け / 仕様整理 / レベル設計

概要

敵に見つかる前にすべてを終わらせるステルスゲーム。プレイヤーは盤面を反転しながら進み、敵の視界を避けて安全なルートとタイミングで標的を仕留める。反転によって安全地帯と危険地帯が入れ替わるため、行動前の判断が重要になる。

企画意図

状況が変化する中で最適な行動を考える、思考型ステルス体験を目指した。

工夫した点

  • 盤面反転によって安全と危険が入れ替わる構造
  • 一手ごとの判断が重要になる設計

作品 03

てるてるウォーズ

モード切替と多様な効果で守り抜くタワーディフェンス。

ジャンル：タワーディフェンス 制作人数：4人 担当：企画 / タスク振り分け / 仕様整理 / レベル設計 / 素材集め / 雑用

概要

てるてる坊主には「通常」と「逆さ」の2つのモードがあり、それぞれ異なる性能を発揮。スロー・毒・スタン・ノックバックなど10種類の特殊能力を駆使して、制限時間まで耐えるタワーディフェンスゲーム。

企画意図

モード切替と複数の能力を組み合わせることで、戦略的に考えるタワーディフェンス体験を目指した。

工夫した点

  • 通常と逆さモードによる性能変化
  • 複数の状態異常を組み合わせた戦略性

作品 04

V-Link Battle

配信文化そのものを戦いに変える、Vtuber対戦アクション。

ジャンル：3D対戦アクション 制作人数：8人 担当：企画 / タスク振り分け / 仕様整理 / システム設計 / 雑用

概要

Vtuberをモチーフにした3D対戦アクションゲーム。通常攻撃やガードに加え、「アピール」によってボルテージを高めながら戦う独自システムを採用。コメントや通知ウィンドウなど、配信文化を取り入れたギミックによって他にはない対戦体験を目指しました。

企画意図

Vtuberの魅力である「視聴者との双方向コミュニケーション」を格闘ゲームに落とし込むことを目標に企画しました。勝敗だけでなく、観客を盛り上げる行動そのものを戦略にすることで、配信文化らしい駆け引きを生み出しています。

工夫した点

  • アピールと攻撃を組み合わせた独自のボルテージシステムを設計
  • コメントや通知ウィンドウを戦闘ギミックとして活用
  • 「魅せる行動」が強さにつながるルールを構築
  • 配信文化と格闘ゲームを融合し、新しい対戦体験を目指した
